🚫 知乎软广过滤器

基于智谱 GLM 大模型，自动识别并隐藏知乎 Feed 中的软文广告

① 工具能做什么

  • 扫描知乎首页、推荐、回答列表里的每条帖子，交给大模型判断是否为软文广告（种草、带货、品牌推广）。
  • 判定为软广的帖子会被折叠，原位置留一条黄色提示条，支持一键展开查看或重新折叠。
  • 每条帖子右上角显示识别状态徽标，等待中 / 识别中 / 非软广 / 软广 / 已取消 / 失败，让你清楚知道插件在干嘛。
  • 结果本地缓存，同一条帖子不会重复调用 API。

② 实现原理

页面加载 / 滚动→ 抓取 Feed 节点→ 提取标题 + 开头 200 字→ 串行调用 GLM API→ 返回 is_ad→ 折叠或放行
  • Content Script：监听 DOM 变化，用 MutationObserver 捕获新加载的帖子。
  • 全局串行队列：一次只处理一条，避免并发请求压垮免费额度；用户划得太快、滑出视窗的帖子会自动取消。
  • Service Worker：统一调用智谱 glm-4-flash 免费模型，带内存 + 本地双层缓存。
  • Prompt：系统提示词里描述了软广的典型特征（种草、第一人称体验、品牌植入等），要求模型返回 {"is_ad": true/false, "reason": "..."}。

③ 申请智谱 API Key（免费）

  1. 打开 https://open.bigmodel.cn，使用手机号 / 微信注册登录。
  2. 在控制台左侧菜单找到「API Keys」或「密钥管理」。
  3. 点「创建 API Key」，复制生成的字符串（形如 xxxxxxxx.yyyyyy）。
  4. 新用户会自动获得免费 Token，glm-4-flash 本身是完全免费的模型，够日常使用。
💡 Key 只保存在你本地浏览器（chrome.storage.sync），不会上传到第三方。

④ 配置插件

  1. 点击 Chrome 右上角的插件图标 🚫。
  2. 把刚才复制的 API Key 粘贴到输入框，点「保存」。
  3. 插件会自动测试 Key 是否可用，通过后显示 ✓ 已保存。
  4. 打开 知乎，开始愉快刷帖 🎉。

⑤ 状态徽标说明

⏳ 等待中 已入队，排队等待处理
🔄 识别中 正在请求 GLM API
✓ 非软广 正常帖子，已放行
🚫 软广 判定为软广，已折叠
⊘ 已取消 你滑过去了，跳过
○ 未识别 无可提取内容
⚠ 失败 API 或网络错误

⑥ 常见问题

  • 会不会误判？ 会。判定基于模型对标题和开头 200 字的理解，边缘案例可能判错，点提示条上的「显示」即可还原。
  • 隐私？ 只把标题和摘要前 200 字发给智谱 API，不上传 cookie、账号、正文全文，也不经过任何第三方服务器。
  • 看不到效果？ 打开插件看统计是否在增长；F12 打开 Console 看有无错误；检查 API Key 是否保存成功。
  • 原生广告？ 本插件不处理知乎自带的「推广」标签广告，那种请用 AdBlock 类插件屏蔽。本插件专注识别伪装成正常帖子的软文。
